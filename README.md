# OpenVoice IIITM

OpenVoice IIITM is a secure, college-only discussion platform for IIITM students.

## Features

- **Authentication:** Google OAuth restricted to `@iiitm.ac.in` emails.
- **Posting:** Create, view, and read threaded discussions.
- **Reactions:** Like and Dislike posts.
- **Reporting:** Robust reporting and moderation system.
- **Search:** Full-text search for posts.
- **Tech Stack:** Next.js 14, TypeScript, Prisma, PostgreSQL, Tailwind CSS.
- 
https://openvoice-production.onrender.com
## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
